Cookie:

  • If you don’t select “Remember me”, your device will forget your account name when you reload this page — then you can Start a new account, and play again. The abandoned account remains on the Corda node, but with a random name so nobody knows it was yours.

  • If you select “Remember me”, a cookie is created on this device to remember the account name — then when you reload the page you can continue to use the same account.

    I think that according to the GDPR this would be an essential session cookie, not a “preferences” cookie — so we needn’t (and won’t) “Document and store” your consent for this cookie — but we won’t even create it unless you select “Remember me”.

    The session cookie will expire after days.
